# Machine Learning Fundamentals

## What is Machine Learning?

Machine Learning is a subset of AI where systems learn from data rather than being explicitly programmed.

## Types of Learning

### Supervised Learning
- Training data has labeled outputs
- Examples: Classification, Regression
- Use case: Predicting house prices

### Unsupervised Learning
- Training data has no labels
- Examples: Clustering, Dimensionality Reduction
- Use case: Customer segmentation

### Reinforcement Learning
- Agent learns by rewards and penalties
- Examples: Game AI, Robot control
- Use case: Autonomous vehicles

## Key Concepts

- **Dataset**: Collection of training examples
- **Features**: Input variables
- **Labels**: Target outputs (supervised)
- **Model**: Mathematical function that maps inputs to outputs
- **Training**: Process of adjusting model parameters
- **Evaluation**: Measuring model performance

## Common Algorithms

- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)
- K-Means Clustering
- Neural Networks
